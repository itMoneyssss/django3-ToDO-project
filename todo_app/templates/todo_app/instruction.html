Instruction:

  1. If you want to create a new task, click on the "Create" button on the sidebar to go to the task creation page. Fill in the required fields and click the "Save" button to create a task.
  2. If you want to view a list of all current tasks, click on the "Home" button on the sidebar.
  3. If you want to edit a task, click on its name in the task list to go to the edit page. Make the necessary changes and click the "Save" button to save the changes.
  4. If you want to delete a task, click on its name in the task list to go to the edit page, and then click the Delete button to delete the task.
  5. If you want to mark a task as completed, click on its name in the task list to go to the edit page, and then click the "Completed" button to complete the task.
  6. If you want to view detailed information about a task, click on its name in the task list to go to the task details page. Here you can see the name, description and status of the task.
  7. You can use the search on the main page to find tasks by keyword or phrase. Enter a keyword or phrase in the search box and click the "Search" button.
  8. If you want to make the task important, then click on the importance button and when you save it, it will be highlighted in red
